创建一个 b.js 导出两个值:
```js
export const name  = 'chen';
export const age = 21;
```

两种使用方法:
```js
// 方法一: 会导出这个默认的对象作为一个对象
import {name, age } from './b.js'
console.log(name) 

// 方法二: 会将若干export导出的内容组合成一个对象返回；
import * as b from './b.js'
console.log(b.name)
console.log(b.age)
```
